What is an "EOL sequence"?

  • "EOL" stands for "end of line"
  • "EOL sequence" is a sequence of hidden characters that splits a line of text

Tell Me More

Words normally appear next to each other on the same line. For example:

Apple Banana Carrot

We can split this line into three lines by putting an EOL sequence between each word:

Apple
Banana
Carrot

The EOL's are hidden. If you could see the EOL'S, it would look like this:

Apple(EOL)
Banana(EOL)
Carrot
