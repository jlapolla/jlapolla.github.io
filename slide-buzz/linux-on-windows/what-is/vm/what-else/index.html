Everything you can do with a real computer, and more

Things you can do with a real computer AND a virtual machine:

  • Install an operating system
  • Install programs
  • Partition the hard disk
  • Add RAM
  • Change a hardware component

Things you can ONLY do with a virtual machine:

  • Back up a virtual machine
    • Remember, a virtual machine is just a file on your computer
  • Run multiple virtual machines at the same time
  • Send a virtual machine to someone else
